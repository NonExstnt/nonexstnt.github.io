COS40007 Design Project

Description: A machine learning-based web application for predicting 5G network performance.

Key Features:

  • 5G Bitrate Predictor using LSTM (PyTorch)
  • 5G Position Predictor using XGBoost
  • FastAPI backend serving ML models
